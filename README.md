DRONIGAMI_V2.0
Install Instructions
Type "cd C:/" and hit enter. If this does not work, try "cd d/ C:/"
Type "git clone https://github.com/rrhee-MOTF1/DRONIGAMI_V2.0.git" and hit enter
Navigate to "C:\DRONIGAMI_V2.0" in Windows File Explorer
Double click on "install" to run the install batch file
DRONIGAMI V2.0 should have added a new dronigami shortcut to your desktop with a gray origami crane icon. Double click to start the app
The app is set to run on 127.0.0.1:8080 (localhost port 8080)
